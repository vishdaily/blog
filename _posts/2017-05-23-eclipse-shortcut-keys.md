## Changing existing shortcut keys

<p style="padding-left: 30px;">
  <strong>Mac</strong>
</p>

<p style="padding-left: 30px;">
  Eclipse -> Preferences -> Keys
</p>

<p style="padding-left: 60px;">
  Search for Previous Tab -> Command + Shift + <-<br /> Select When as -> Editing Java Resource<br /> Apply
</p>
